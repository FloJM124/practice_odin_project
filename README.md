## HTML 
Stage one of this project introduces the fundamentals of web development using only **HTML**. 

- This project helps you practice semantic HTML, anchor-link navigation, and clear content organization.
- The homepage introduces the website and links to three recipe web pages.
- The project demonstrates how to use heading elements: `h1` for page titles, `h2` for section headings, and `h3` for subsection headings.
- To link between web pages, use the `anchor` element with the `href` attribute to specify the destination.
- To display images and captions, use the `figure` element as a container and the `figcaption` element for descriptions or photographer attribution.
- The `image` element uses the `src` attribute to specify the image file, the `alt` attribute to describe the image for accessibility and fallback purposes, and the `width` and `height` attributes to prevent layout shifts.
- The `strong` element indicates importance, and the `em` element indicates emphasis, both of which improve accessibility for screen readers.
- `Lorem Ipsum` text can be placed inside paragraph elements to generate placeholder content. 

## CSS
Stage two focuses on styling the website structure with **CSS**.

- Using an external stylesheet is a best practice because it provides greater flexibility and allows multiple web pages to share the same styles.
- CSS can change how HTML elements behave in normal flow by using properties such as `display`.
- Understanding the box model is important because it explains how elements are structured individually and how they interact with one another.
- You can use DevTools to inspect the box model and view the styles applied to each HTML element. Simply select an element and inspect it.
