# Odin Recipes

A simple recipe website built as part of the **The Odin Project** Foundations curriculum. This project demonstrates basic HTML structure, linking between pages, and deploying a static site using GitHub Pages.

## Live Demo
https://FloJM124.github.io/odin-recipes

## Project Structure
- `index.html` - Main page listing all recipes
- `recipes/pancakes.html`
- `recipes/chicken-adobo.html`
- `recipes/fried-rice.html`

## What I Learned
- Structuring a basic HTML document
- Creating and organizing multi-page websites
- Using relative links between pages
- Initializing a Git repository and making commits
- Pushing code to GitHub using SSH
- Deploying a site with GitHub Pages
